# Phase 1 — Data Collection

Download all Islamic source data needed for training.

---

## What This Phase Does

| Script | What it downloads | Output |
|--------|------------------|--------|
| `download_quran.py` | Full Quran (Arabic + English + Tafsir) | `raw/quran/quran_full.json` |
| `download_hadith.py` | 6 Hadith collections (Arabic + English) | `raw/hadith/*.json` |
| `download_hadith.py` | 99 Names of Allah | `raw/supplementary/asmaul_husna.json` |
| `validate_sources.py` | Validates everything above | `logs/validation_{timestamp}.txt` |

## Step-by-Step Instructions

### Step 1 — Download Quran

```bash
cd 01_data_collection/scripts

python download_quran.py
```

This downloads:
- Arabic text (Uthmani script) for all 6,236 ayahs
- Sahih International English translation
- Ibn Kathir Tafsir (English summary per ayah)

Expected output:
```
Fetching chapter list...
Fetched metadata for 114 chapters
Downloading verses and tafsir for all 114 surahs...
Surahs: 100%|████████| 114/114 [12:30<00:00]
Saved: .../raw/quran/quran_full.json (45.2 MB)
Total surahs: 114 | Total ayahs: 6236
Ayah count verified: 6236 / 6236
Completed in 750.3 seconds
```

**Options:**
```bash
# Skip tafsir (faster, ~5 min instead of ~12 min)
python download_quran.py --skip-tafsir

# Custom output directory
python download_quran.py --output-dir /your/path

# Debug logging
python download_quran.py --log-level DEBUG
```

## Troubleshooting

**Connection errors during Quran download:**
```bash
# Run again — the script has 3-attempt retry with backoff
python download_quran.py
```

**Rate limiting (HTTP 429):**
The script handles this automatically with exponential backoff.
If persistent, increase `REQUEST_DELAY` at the top of the script.

**Tafsir coverage below 80%:**
Some ayahs may not have tafsir in the database.
This is normal — the training will use an empty string for those.

**Hadith count below expected:**
The fawazahmed0 API is the source of truth. Counts may differ slightly
from classical references due to numbering conventions.
The validation uses minimum thresholds, not exact targets.
